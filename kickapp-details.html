Project Description

KickApp is a social media app where users can chat with each other and share media posts and files.

Tech stack:
Java and Android Framework for android app development
Swift for IOS app development
PHP for Back-end
Google FCM for in-app notifications
FFMPEG for video editing within the app


My role:

I was working with a team of 3 android app developers. Some of my contributions are as follows:
- Integrated and implemented Google Firebase and push notifications functionality with in the app.
- Developed chat system where users can share text messages and files
- Integrated FFMPEG within the app to edit videos
- Contributed a bit in back-end PHP to develop/modify APIs
