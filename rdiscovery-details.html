Project Description

R Discovery provides a one-stop personalized academic library that hosts content from Microsoft Academic, PubMed and PubMed Central and several other sources to help you catch up on literature reading.
RDiscovery has over 100M+ scholarly articles, 14M+ authors and 32K+ journals to search articles from.

Tech stack:
React Native for android and ios app development
Laravel framework for Back-end
MySQL as a database
AWS for hosting
Angular.js, HTML, CSS, JS for website
