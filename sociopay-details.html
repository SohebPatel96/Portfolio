Project Description

SocioPay is a UAE-based payment app & digital wallet that is powered by noqodi. It is the first payment & retail app in the UAE with the concept of charitable giving and heavily rewards you for the same too.

Tech stack:
React Native for android and ios app development
Laravel framework for Back-end
MySQL as a database
Azure for hosting
Vue.js, HTML, CSS, JS for B2C and B2B website


My role:

I was working as a Full stack web developer. I have worked on the back-end APIs that are powering the Mobile apps and websites.
Worked on integrating UAE payment service 'Noqodi', to enable customers to make payment via Card/Wallet and Netbanking
Worked with a team to develop customer website to display merchants nearby.
Designed and developed merchant web application for merchants to view transactions, raise issues, generate payment links etc
