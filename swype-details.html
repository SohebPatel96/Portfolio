Project Description

Swype is a staff sign-in and visitor management system. Swype signing-in system gives quick and accurate visitor and staff information along with payroll calculations, staff working hours, leave and much more.

Tech stack:
Java and Android framework for android app development
Laravel framework for Back-end
MySQL as a database
Windows Server for hosting
C# for windows desktop application
Vue.js, HTML, CSS, JS for web application


My role:

I was working as a Full stack web developer. I have worked on the back-end APIs that are powering the Mobile apps, web application and windows desktop application

I have integrated Verilook's face recognition technology to clock in and clock out visitors and staff members.

Worked on developing web application to view staff sign in/out, attendance & leaves and payable amount and much more.
